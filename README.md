# CommentFocus

# 精选评论插件

---

## 简介

**CommentFocus** 是一款为 Typecho 开发的精选评论管理插件。

在后台评论列表增加「设为精选」按钮，一键标记优质评论；前台通过短代码 `[featured_comments]` 展示所有精选内容，自动适配手机端与 PC 端。

---

## 功能特性

- 后台一键精选 / 取消精选评论
- 前台短代码调用，零代码侵入主题
- 支持 4 种排序方式（精选时间 / 评论时间）
- 可选显示来源文章标题，点击直达原文
- 自适应布局，手机端与 PC 端独立优化
- 支持自定义 CSS 覆盖默认样式
- 头像自动走 Cravatar 加速

---

## 安装方法

### 方式一：手动上传

1. 下载本插件，解压后重命名为 `CommentFocus`
2. 将文件夹上传至 `usr/plugins/` 目录
3. 进入 Typecho 后台 → 控制台 → 插件 → 启用「精选评论插件」


然后在后台启用即可。

---

## 使用说明

### 后台精选评论

启用插件后，进入 **后台 → 管理 → 评论**，每条评论的操作列会出现：

- **设为精选** —— 点击后该评论被标记为精选
- **已精选** —— 再次点击可取消精选

精选状态实时保存，无需刷新页面。

### 前台展示精选

在任意文章或独立页面中插入短代码：

```
[featured_comments]
```

发布后，该位置会自动渲染精选评论列表。

---

## 插件配置

进入 **后台 → 控制台 → 插件 → 精选评论插件 → 设置**：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 启用插件 | 总开关，关闭后前台不再渲染 | 启用 |
| 按钮文字 | 后台评论列表按钮显示文案 | 设为精选 |
| 显示数量 | 前台最多展示几条精选评论 | 10 |
| 前台排序方式 | 控制精选列表的排列顺序 | 最新精选在前 |
| 显示文章标题 | 是否在评论上方显示来源文章 | 显示 |
| 自定义 CSS | 直接覆盖前端默认样式 | 留空 |

---

## 排序方式详解

| 选项 | 排序逻辑 |
|------|----------|
| 最新精选在前 | 后设为精选的评论排在最前面 |
| 最早精选在前 | 先设为精选的评论排在最前面 |
| 最新评论在前 | 按评论本身发表时间，新的在前 |
| 最早评论在前 | 按评论本身发表时间，旧的在先 |

---

## 自定义样式

插件自带自适应默认样式，如需微调，在 **自定义 CSS** 框中写入代码即可直接覆盖。

示例：

```css
.featured-comments {
    background: #fafafa;
    border-radius: 12px;
}
.fc-author {
    color: #333;
    font-weight: 600;
}
```

保存后前台即时生效。
